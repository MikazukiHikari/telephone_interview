# 衛生局邀約民眾健檢難易度評估 
與衛生局合作，透過他們所提供民眾篩檢資料以及電話邀約資料，設計一套分類流程、模型將原先需要透過打電話處理的資料轉變成幫他們區分以及預測出這些人中哪些適合透過電話邀約哪些適合用簡訊通知，以此減輕他們的人力、時間以及金錢成本。
